# BroomDash — Unity 2D

Inspired by Jetpack Joyride with custom mechanics.

## How to Play
- **Space** — Jetpack thrust
- **Space + Shift** — Boost (drains gauge)
- **S / Down** — Fast land
- Survive as long as possible and beat your best score
## Setup Instructions
1. Clone the repo
2. Open in Unity 6000.3.10f1 or later
3. Open `Assets/Scenes/` and load the  scene
4. Press Play
